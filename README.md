# Dubai Real Estate Intelligence Dashboard

## Project Overview

This project presents an **interactive Power BI dashboard** designed to analyze the Dubai real estate market.
The dashboard provides insights into **property prices, listings distribution, developer activity, and market trends across locations**.

The main objective of this project is to help understand **how property type, location, and property size influence real estate prices in Dubai**.
By using visual analytics, the dashboard helps users quickly identify **high-value locations, price trends, and property distribution patterns**.

This project demonstrates skills in **data analysis, dashboard development, and business intelligence using Power BI**.

---

# Dataset Description

The dataset contains information about property listings in Dubai, including pricing, location, property characteristics, and developers.

### Key Fields in the Dataset

* **Location / Neighborhood** – Area where the property is located
* **Property Type** – Type of property (Apartment, Villa, etc.)
* **Bedrooms** – Number of bedrooms in the property
* **Bathrooms** – Number of bathrooms
* **Developer** – Company or developer responsible for the property
* **Listing Category** – Property category (for example: sale or rent)
* **Price** – Property price
* **Size (Square Feet)** – Total property area
* **Price per Square Foot** – Cost per square foot
* **Date** – Used to analyze price trends over time

# Dashboard Visualizations

The dashboard includes several visual elements to analyze the Dubai real estate market.

---

### 1️⃣ Average Price per Location

**Chart Type:** Map Visualization

This map shows the **average property price by location** across Dubai.
It helps identify **high-value neighborhoods and premium real estate areas**.

---

### 2️⃣ Price by Property Type

**Chart Type:** Clustered Bar Chart

This chart compares the **average price of different property types**, such as apartments or villas, helping understand which types are more expensive.

---

### 3️⃣ Listings by Bedrooms (Colored by Property Type)

**Chart Type:** Stacked Column Chart

Displays the **number of property listings based on bedroom count**, categorized by property type.

This helps identify common property configurations in the market.

---

### 4️⃣ Price Trend Over Time

**Chart Type:** Line Chart

Shows how property prices have changed **over time**, helping analyze **market trends and price fluctuations**.

---

### 5️⃣ Listings by Developer and Property Type

**Chart Type:** Heatmap

The heatmap visualizes the **distribution of property listings across developers and property types**, helping identify major developers and their property offerings.

---

# Key Performance Indicators (KPIs)

The dashboard includes important KPI metrics for quick insights:

* **Total Listings** – Total number of properties available in the dataset
* **Average Property Price** – Average price of listed properties
* **Highest Price Property** – Maximum property price in the dataset
* **Average Property Size (Square Root / Square Feet)** – Average property size
* **Average Price per Square Foot** – Cost efficiency of properties

These KPIs provide a **quick summary of the Dubai real estate market**.

---

# Interactive Filters (Slicers)

The dashboard includes slicers that allow users to interact with the data:

* **Neighborhood Filter** – Analyze specific locations
* **Bedrooms Filter** – Explore properties based on bedroom count
* **Bathrooms Filter** – Filter listings by number of bathrooms
* **Listing Category Filter** – Analyze properties based on listing type

These filters make the dashboard **interactive and flexible for analysis**.

---

# Key Insights

The dashboard helps uncover several important real estate insights:

* Certain **neighborhoods in Dubai have significantly higher average property prices**.
* **Property type influences price levels**, with some property types commanding higher values.
* Listings are often concentrated around **specific bedroom configurations**.
* Major developers dominate listings in certain property categories.
* Property prices show **trend patterns over time**, reflecting market movement.


# How to Use This Project

1. Download the repository.
2. Open the **Power BI (.pbix) file** using Microsoft Power BI Desktop.
3. Explore the interactive dashboard.
4. Use slicers to filter by **neighborhood, bedrooms, bathrooms, or listing category**.
