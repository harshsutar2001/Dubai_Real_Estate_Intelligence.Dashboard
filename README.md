# Dubai_Real_Estate_Intelligence.Dashboard

Dubai Real Estate Intelligence Dashboard

Tools Used:
Power BI – For interactive dashboard and visualizations
Power Query – For data cleaning, transformation, and enrichment
CSV – Source data preparation

Project Overview
The Dubai Real Estate Intelligence Dashboard is an interactive and visually appealing Power BI project designed to provide actionable insights into Dubai's property market.

Stakeholders:

Investors
Real estate agents
Developers
Policymakers

Purpose: Enable stakeholders to make data-driven decisions based on real-time property trends, pricing, and distribution.

Objective

Design a professional and interactive dashboard highlighting key metrics in Dubai’s real estate sector
Help stakeholders identify investment opportunities and market trends efficiently
Facilitate strategic decision-making for property purchases, sales, and development planning

Dataset Summary

The dataset includes:
Core Features: Location, Price, Bedrooms, Bathrooms, Size (sqft)
Property Details: Property Type, Developer, Year Built, Furnishing Status

Enriched Columns:

Price per Sqft: Price ÷ Size
Property Age: 2025 − Year Built
Listing Category: High-End / Mid-Range / Budget (based on price quantiles)
This enriched data allows for deeper insights into property value trends, developer performance, and property segmentation.

Data Preparation (Power Query)

Data Cleaning: Removed missing and incorrect values
Duplicate Removal: Eliminated repeated entries
Data Type Conversion: Numeric and categorical fields formatted correctly

Calculated Columns:

Price per Sqft = Price ÷ Size
Property Age = 2025 − Year Built
Listing Category = High-End, Mid-Range, or Budget
Ensures the data is consistent, reliable, and ready for visualization

Dashboard KPIs

Total Listings – Overall number of properties in the dataset
Average Price – Mean property price
Average Size (sqft) – Typical property size
Highest Priced Property – Highlights premium listings
Average Price per Sqft – Understand property value relative to size

Visual Insights

Map Visualization: Average Price per Location – identify high-value areas
Bar Chart: Price distribution by Property Type
Stacked Column Chart: Listings by Bedrooms & Property Type
Tree Map: Listings by Developer – compare developer activity
Line Chart (Optional): Price trend over time
Interactive visuals allow stakeholders to explore patterns, compare segments, and analyze property dynamics in Dubai efficiently.

Key Takeaways

Investors can identify high-value locations and trending property types
Agents can analyze developer activity and pricing trends
Policymakers can monitor market distribution and housing affordability
Enables data-driven strategic planning for property investments and development
Provides visual, interactive insights to simplify complex real estate data
Helps identify emerging neighborhoods with high growth potential
Supports risk assessment by highlighting overpriced or underdeveloped areas
Encourages transparency in the real estate sector, benefiting all stakeholders

Conclusion:

The Dubai Real Estate Intelligence Dashboard combines data preparation, enriched metrics, and interactive visualizations to provide a comprehensive and actionable overview of Dubai’s real estate market.
